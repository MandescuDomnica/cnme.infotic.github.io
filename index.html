TIC

Teorie și aplicații

TIC este un subiect larg, iar conceptele evoluează. Acesta acoperă orice produs care va stoca, prelua, manipula, transmite sau primi informații electronic într-o formă digitală.


INFO

Limbajul de programare este un limbaj formal de expresii și reguli valide de formulare a instrucțiunilor pentru un computer. El permite programatorului să specifice acțiunile pe care trebuie să le execute calculatorul.


CNME Centru de examinare ECDL-RO418

ECDL/ICDL este un program de certificare a competențelor digitale recunoscut la nivel internațional. Permisul ECDL/ICDL oferă o bază standard de cunoștințe IT utile în societatea informațională actuală.

Materiale elevi cu CES

CES se referă la resurse și metode adaptate, prin care elevii primesc sprijin personalizat, materiale accesibile și un ritm de lucru potrivit nevoilor lor.
